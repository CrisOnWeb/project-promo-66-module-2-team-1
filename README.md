# Refugio Adalab — Generador de Tarjetas de Adopción

Aplicación web que permite crear una tarjeta de adopción para mascotas a partir de un formulario dinámico, con previsualización en tiempo real y persistencia de datos en el navegador.

Este proyecto ha sido desarrollado en equipo como parte del Módulo 2 (JavaScript) del bootcamp de desarrollo web.

---

## Funcionalidades

- Personalización de la tarjeta mediante paletas de color.
- Formulario con información de la mascota.
- Previsualización automática en tiempo real.
- Subida de imagen.
- Interfaz organizada en secciones colapsables.
- Guardado automático en localStorage.
- Reset del formulario.

---

## Tecnologías utilizadas

- HTML5
- SASS (SCSS)
- JavaScript (ES6)
- Vite (Adalab Web Starter Kit)
- FontAwesome

---

## Instalación y uso

1. Clonar el repositorio:

```bash
git clone https://github.com/CrisOnWeb/project-promo-66-module-2-team-1.git
```

2. Instalar dependencias:

```bash
npm install
```

3. Arrancar el proyecto:

```bash
npm run dev
```

Abrir en el navegador: http://localhost:5173/

4. Generar build de producción:

```bash
npm run build
```
